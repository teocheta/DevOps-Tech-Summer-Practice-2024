* vim

    - in order to quit: ESC + :q
    - quit and not save: :q!
    - quit and save: :wq

        -> insert mode
        -> command mode
        -> visual mode

    - readonly mode: view testfile

* less - can go back and forward
* head - se first lines of file (10 by default)
* tail - same but end lines
